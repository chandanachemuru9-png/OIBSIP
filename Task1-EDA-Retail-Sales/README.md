# Task 1 - EDA on Retail Sales Data
## OIBSIP Internship - Data Analytics

### Overview
Exploratory Data Analysis on a retail sales dataset 
containing 1000 transactions.

### Dataset Features
- Transaction_ID, Date, Customer_ID
- Gender, Age, Product_Category
- Quantity, Price_per_Unit, Total_Amount

### Analysis Performed
- Sales by Product Category
- Sales distribution by Gender
- Age distribution of customers
- Monthly sales trend
- Correlation heatmap

### Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn

### Key Findings
- Food category has highest total sales
- Sales are nearly equal across genders
- Customer age ranges from 18 to 65
